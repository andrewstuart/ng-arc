withService
directive in module ng-arc

Description

Injects a service into a new scope the DOM element.

Usage

as attribute
<ANY with-service="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
withServiceString

The name of the service to inject. May be an expression "X as Y" in which case the X service will be exposed by the name Y.
